# Company Logos

This folder contains company logos for the instructor's professional experience section on the About Instructor page.

## Required Logos

Add PNG or SVG files for the following companies:

1. **honeywell-logo.png** - Honeywell International
2. **samsung-logo.png** - Samsung Electronics
3. **aristocrat-logo.png** - Aristocrat Technologies
4. **incedo-logo.png** - Incedo Inc.
5. **hcl-technologies-logo.png** - HCL Technologies

## File Specifications

- **Format**: PNG or SVG (SVG preferred for scalability)
- **Resolution**: 300x200px or higher (for PNG)
- **Background**: Transparent (PNG) or white (SVG)
- **Style**: Company official logo, white or dark variants
- **File Size**: Keep under 100KB per file

## How to Add Logos

1. Download the official logos from each company's website or brand resources
2. Resize/optimize them to the specifications above
3. Place them in this `logos` folder
4. Ensure filenames match exactly:
   - `honeywell-logo.png`
   - `samsung-logo.png`
   - `aristocrat-logo.png`
   - `incedo-logo.png`
   - `hcl-technologies-logo.png`

## Display Location

These logos are displayed on the About Instructor page in the "Professional Experience" section, showing Amit Lamba's corporate background across these five major technology companies.

## Styling

- Logos are displayed in a 5-column grid on desktop (2-column on mobile)
- Hover effect: Cards lift up with a shadow
- Container height: 80px with centered, contained logo
- Dark mode support included

## Notes

- If you have high-res vector files (.eps, .ai), convert to SVG or PNG
- Ensure logos are official company marks for professional appearance
- Test on both light and dark theme modes
